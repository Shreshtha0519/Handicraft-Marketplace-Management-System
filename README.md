# Handicraft Marketplace Management System

This is a simple Java Maven project for managing products in a handicraft marketplace.

## Tech Stack
- Java
- Maven
- JUnit

## Project Structure
- `handicraft-marketplace/` - main Maven module
- `.github/workflows/` - GitHub Actions CI workflow

## Run Locally
1. Open terminal in `handicraft-marketplace`.
2. Build the project:
	`mvn clean package`
3. Run tests:
	`mvn test`

## Notes
This project includes a CI workflow that runs build and test steps on GitHub.
